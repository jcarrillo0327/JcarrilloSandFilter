# Carrillo Sand Unit Simulator

Hybrid 3D + engineering simulator for sand filtration in flowback operations.

Features:
- Swirl induction chamber
- Venturi nozzle acceleration
- Gravity dropout
- 100-mesh sand behavior
- Filter cake buildup model
- Real-time pressure drop graph
- 3D particle visualization
- CSV data export
